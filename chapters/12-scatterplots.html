12  Scatterplots

Previous modules have explored the distribution of a single continuous variable (histograms in Chapter 6), the relationship between continuous and categorical variables (catplots in Chapter 7), and time series as a special kind of continuous data (line plots in Chapter 10). This chapter explores the relationship between two continuous variables and its corresponding visualization: the scatterplot.

Scatterplots are deceptively simple — two axes, a cloud of dots — but the choices you make about size, color, transparency, annotation, and axis scales can dramatically affect what the reader sees and concludes.

12.1 Setup

import numpy as np

import pandas as pd
idx = pd.IndexSlice
pd.options.display.max_columns = 200

%matplotlib inline
import matplotlib.pyplot as plt
from matplotlib import ticker

import seaborn as sb
from scipy import stats

12.2 The Data

We will work with a dataset of U.S. counties combining demographic, economic, political, and health variables. In the Appendix of the companion repository, you can see how several county-level datasets were joined together. The data includes 2020 Census estimates (source) about social characteristics (DP02 table), economic characteristics (DP03 table), and demographic characteristics (DP05 table). It also includes the share of votes cast for Donald Trump in the 2016 and 2020 elections (source) and cumulative COVID-19 case and death counts from The New York Times (source). See Chapter 15 for full provenance details.

counties_df = pd.read_csv('us_counties.csv', dtype={0: str}, na_values=['-'])
counties_df.rename(columns={'Unnamed: 0': 'FIPS'}, inplace=True)
counties_df.set_index('county_fips', inplace=True)
print(counties_df.shape)
counties_df.head()

Let’s filter to just Colorado’s 64 counties for initial exploration.

co_counties = counties_df.copy()[counties_df['state'] == 'Colorado']

print(co_counties.shape)
co_counties.head(8)
biggest_co_counties = co_counties.sort_values('Population', ascending=False)['county'].head(10).values
biggest_co_counties

It is important to emphasize that counties are units of approximately equal geographic size, not equal population. Most Colorado counties have small populations.

ax = co_counties['Population'].hist(bins=np.linspace(0, 8e5, 25), figsize=(8, 4))
ax.set_ylim((0, 50))
ax.grid(None)
ax.set_xlabel('Population')
ax.set_ylabel('Counties')
ax.xaxis.set_major_formatter(ticker.StrMethodFormatter('{x:,.0f}'))

12.3 Making Scatterplots

There are three common interfaces for creating scatterplots in Python. All produce similar results; choose whichever syntax you find most readable.

12.3.1 matplotlib

f, ax = plt.subplots(figsize=(6, 6))

ax.scatter(
    x=co_counties['Median_Income'],
    y=co_counties['covid_deaths_percapita']
)

ax.set_xlabel('Median Income ($)')
ax.set_ylabel('COVID-19 Deaths per Capita')

12.3.2 pandas

f, ax = plt.subplots(figsize=(6, 6))

co_counties.plot.scatter(
    x='Pct_Broadband',
    y='covid_cases_percapita',
    ax=ax
)

12.3.3 seaborn

f, ax = plt.subplots(figsize=(6, 6))

sb.scatterplot(
    x='Pct_Broadband',
    y='covid_cases_percapita',
    data=co_counties,
    ax=ax
)

12.4 Customizing Scatterplots

12.4.1 Point size

Change the size of all points with the s parameter.

f, ax = plt.subplots(figsize=(6, 6))

co_counties.plot.scatter(
    x='Median_Income',
    y='covid_deaths_percapita',
    s=5,
    ax=ax
)

Map a third variable to point size to create a bubble chart. This is useful for representing a quantity like college graduation rates without requiring a third axis.

f, ax = plt.subplots(figsize=(6, 6))

co_counties['size'] = co_counties['Pct_College_Grad'] ** 1.5

co_counties.plot.scatter(
    x='Median_Income',
    y='covid_deaths_percapita',
    s='size',
    ax=ax
)

12.4.2 Transparency

When points overlap heavily, setting alpha to a low value (0.1–0.3) reveals the density of the point cloud.

f, ax = plt.subplots(figsize=(6, 6))

co_counties.plot.scatter(
    x='Median_Income',
    y='covid_deaths_percapita',
    s='size',
    ax=ax,
    alpha=.5
)

12.4.3 Color

Map a variable to color to add a fourth dimension. The pandas interface uses c for a continuous color variable.

ax = co_counties.plot.scatter(
    x='Population',
    y='trump_2020'
)

ax.set_xscale('log')

With seaborn, use hue for the color variable and size for the size variable.

f, ax = plt.subplots(figsize=(6, 6))

sb.scatterplot(
    x='Pct_College_Grad',
    y='Median_Income',
    hue='trump_2020',
    size='covid_deaths',
    palette='coolwarm',
    data=co_counties,
    ax=ax
)

12.4.4 Annotations

For small datasets, labeling individual points provides rich context. Loop through the data and use ax.annotate() to place text near each point.

f, ax = plt.subplots(figsize=(6, 6))

co_counties.plot.scatter(
    x='Pct_College_Grad',
    y='Median_Income',
    alpha=.75,
    ax=ax
)

for fips, name in co_counties['county'].to_dict().items():
    if name in biggest_co_counties:
        x = co_counties.loc[fips, 'Pct_College_Grad']
        y = co_counties.loc[fips, 'Median_Income']
        ax.annotate(name, (x, y), fontsize=7, alpha=.8)

12.5 Conventions for Causal Arguments

Of course correlation is not causation, but whenever you make a scatterplot, you are making an implicit causal argument: one variable is influencing the other. The dominant convention is to place the presumed cause on the x-axis and the presumed effect on the y-axis.

If you are arguing that broadband connectivity influences COVID exposure behavior, broadband goes on the x-axis and COVID cases on the y-axis.

f, ax = plt.subplots(figsize=(6, 6))
counties_df.plot.scatter(x='Pct_Broadband', y='covid_cases_percapita', ax=ax, alpha=.25)

If you reverse the axes, you are implicitly arguing the opposite causal direction — that COVID exposure influences broadband connectivity — which is much harder to justify theoretically.

f, ax = plt.subplots(figsize=(6, 6))
counties_df.plot.scatter(x='covid_cases_percapita', y='Pct_Broadband', ax=ax)

Be intentional about this choice. Swapping the axes changes the implicit argument.

12.6 Types of Bivariate Relationships

12.6.1 Positive relationship

Counties where more of the population was born in the same state also tend to have higher high school graduation rates. The Pearson correlation coefficient quantifies this linear association.

f, ax = plt.subplots(figsize=(6, 6))
counties_df.plot.scatter(x='Pct_Born_Same_State', y='Pct_HS_Grad', ax=ax)
counties_df.loc[:, ['Pct_HS_Grad', 'Pct_Born_Same_State']].corr()

A correlation of ~0.46 is moderate-to-high for social science data, where relationships are complex and layered.

12.6.2 Negative relationship

Counties with more people on public assistance tend to have lower broadband usage.

f, ax = plt.subplots(figsize=(6, 6))
counties_df.plot.scatter(x='Pct_Assistance', y='Pct_Broadband', ax=ax)
counties_df.loc[:, ['Pct_Assistance', 'Pct_Broadband']].corr()

12.6.3 No relationship

There is no obvious linear relationship between percent female and percent Hispanic.

f, ax = plt.subplots(figsize=(6, 6))
counties_df.plot.scatter(x='Pct_Female', y='Pct_Hispanic', ax=ax)
counties_df.loc[:, ['Pct_Female', 'Pct_Hispanic']].corr()

12.6.4 Non-linear relationship

Income and poverty follow a curve: poverty drops steeply as income rises from low levels, then levels off at high incomes. A linear model would miss the shape of this relationship.

f, ax = plt.subplots(figsize=(6, 6))
counties_df.plot.scatter(x='Median_Income', y='Pct_Family_Poverty', ax=ax)
counties_df.loc[:, ['Median_Income', 'Pct_Family_Poverty']].corr()

12.6.5 Reversals and surprises

Sometimes changing one variable to a similar variable can completely reverse a relationship. Trump 2016 vote share increases with more high school graduates but decreases with more college graduates.

f, axs = plt.subplots(nrows=1, ncols=2, figsize=(12, 6), sharey=True,
                       subplot_kw={'ylim': (0, 100)})

counties_df.plot.scatter(x='Pct_HS_Grad', y='trump_2016', ax=axs[0])
axs[0].set_title('Trump 2016 vote share increases with more HS grads')

counties_df.plot.scatter(x='Pct_College_Grad', y='trump_2016', ax=axs[1])
axs[1].set_title('Trump 2016 vote share decreases with more college grads')

f.tight_layout()
counties_df[['Pct_HS_Grad', 'Pct_College_Grad', 'trump_2016']].corr()

12.7 Pair Plots and Correlograms

With many variables, it helps to see all pairwise relationships at once. Seaborn’s pairplot generates a grid of scatterplots.

select_columns = ['trump_2020', 'covid_cases_percapita', 'Pct_Over_65',
                  'Pct_White', 'Pct_Unemployed', 'Median_Income', 'Pct_English_Only']

sb.pairplot(data=counties_df[select_columns])

Customize to show only the lower triangle and overlay regression lines.

sb.pairplot(
    data=counties_df[select_columns],
    kind='reg',
    corner=True,
    plot_kws={
        'scatter_kws': {'alpha': .1},
        'line_kws': {'color': 'red'}
    }
)

Alternatively, compute the full correlation matrix and display it as a heatmap.

counties_corr = counties_df[select_columns].corr()

f, ax = plt.subplots(figsize=(8, 8))
sb.heatmap(data=counties_corr, square=True, cmap='viridis', vmin=-1, vmax=1, annot=True, fmt='.2f')

Seaborn’s clustermap rearranges the rows and columns so that variables with similar correlations are grouped together.

sb.clustermap(data=counties_corr, cmap='viridis', vmin=-1, vmax=1)

## Pair Plots and Correlograms

With many variables, it helps to see all pairwise relationships at once. Seaborn's `pairplot` generates a grid of scatterplots.

```{python}
select_columns = ['trump_2020', 'covid_cases_percapita', 'Pct_Over_65',
                  'Pct_White', 'Pct_Unemployed', 'Median_Income', 'Pct_English_Only']

sb.pairplot(data=counties_df[select_columns])
```

Customize to show only the lower triangle and overlay regression lines.

```{python}
sb.pairplot(
    data=counties_df[select_columns],
    kind='reg',
    corner=True,
    plot_kws={
        'scatter_kws': {'alpha': .1},
        'line_kws': {'color': 'red'}
    }
)
```

Alternatively, compute the full correlation matrix and display it as a heatmap.

```{python}
counties_corr = counties_df[select_columns].corr()

f, ax = plt.subplots(figsize=(8, 8))
sb.heatmap(data=counties_corr, square=True, cmap='viridis', vmin=-1, vmax=1, annot=True, fmt='.2f')
```

Seaborn's `clustermap` rearranges the rows and columns so that variables with similar correlations are grouped together.

```{python}
sb.clustermap(data=counties_corr, cmap='viridis', vmin=-1, vmax=1)
```
